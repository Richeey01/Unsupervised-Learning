# 🌍 Country Clustering with K-Means

## 📌 Описание проекта
Этот проект использует алгоритм **K-Means (unsupervised learning)** для кластеризации стран по уровню развития на основе социально-экономических показателей.

Модель анализирует данные и автоматически делит страны на 3 группы:
- Developed (развитые)
- Developing (развивающиеся)
- Underdeveloped (слаборазвитые)

---

## 🤖 Используемые технологии
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Flask (для веб-интерфейса)

---

## 📊 Используемые данные
Датасет: **Country-data (Kaggle)**  
Используемые признаки:
- GDP per capita (gdpp)
- Child mortality (child_mort)
- Income
- Life expectancy (life_expec)
- Health spending (health)

---

## ⚙️ Как работает проект
1. Данные загружаются из CSV файла
2. Нормализуются с помощью StandardScaler
3. Алгоритм K-Means делит страны на 3 кластера
4. Результаты визуализируются (график)
5. Определяется лучшая страна в каждом кластере
6. Модель сохраняется и используется для предсказаний
