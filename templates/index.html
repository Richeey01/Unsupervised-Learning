🌍 Country Clustering with K-Means

Unsupervised Learning  |  Dataset: Country-data (Kaggle)

📂 Dataset

167 countries, 5 features selected for training.

gdpp child_mort income life_expec health

🤖 What is Unsupervised Learning?

A type of machine learning where the model finds hidden patterns in data without labels. The algorithm groups similar data points on its own — no correct answers provided.

📋 Project Description

K-Means groups 167 countries into 3 clusters by socioeconomic indicators. Features scaled with StandardScaler.

Cluster 0 – Developing Cluster 1 – Developed Cluster 2 – Underdeveloped
